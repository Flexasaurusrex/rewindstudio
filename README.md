# FLEX — Cultural Infrastructure Studio

Portfolio and partnership page for the REWIND ecosystem.

## Deploy

Push to GitHub → Enable GitHub Pages from Settings → Source: main branch, root.

## Custom Domain

1. Add your domain to the `CNAME` file
2. Update `og:image` and `twitter:image` URLs in `index.html` to use the full domain
3. Configure DNS (CNAME to `<username>.github.io` or A records to GitHub Pages IPs)
